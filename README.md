# WeatherDashboard-React
A working good demo project WeatherDashboard using React, API Calling, UseState hook, React Router, Custom CSS, Inline Style

# Download the project into a folder
# GoTo that folder and open cmd
# In cmd while directory pointing to the project folder run: npm install to update packages
# Once packages are updated run: npm run dev to execute the project successfully.
# The project uses two different free APIs 
  First one to fetch Location Lat and Lon coordinates
  Second to use that Lat and Lon to fetch the current weather details
